# Auto-my-invoice 프로덕트 백로그

> 마지막 업데이트: 2026-03-11
> 우선순위 기준: 사용자 가치 × 수익 임팩트 × 기술 의존성

---

## 우선순위 체계

| 레이블 | 의미 | 기준 |
|--------|------|------|
| **P0** | Must Have | 출시 차단 이슈. 없으면 서비스 불가 |
| **P1** | Should Have | 핵심 경험 완성. 첫 달 내 필요 |
| **P2** | Nice to Have | 경쟁 우위 강화. 분기 내 |
| **P3** | Future | 확장 기능. 6개월+ |

## 크기 추정

| 레이블 | 의미 |
|--------|------|
| **XS** | < 2시간 |
| **S** | 2~4시간 |
| **M** | 4~8시간 (1일) |
| **L** | 1~3일 |
| **XL** | 3~5일 |
