# Auto-my-invoice 기능 명세서

> 이 문서는 Google Docs로 옮겨 팀/이해관계자와 공유하기 위한 기능 명세서입니다.

---

## 1. 제품 개요

**Auto-my-invoice**는 프리랜서와 소규모 사업자를 위한 송장 자동화 SaaS입니다.
PDF 송장을 업로드하면 AI가 데이터를 추출하고, 미수금 리마인더를 자동 발송하며, 결제까지 하나의 흐름으로 연결합니다.

**타겟 사용자**: 프리랜서, 소규모 에이전시, 현장 서비스 사업자
**핵심 가치**: "돈 받는 일에 쓰는 시간을 0에 가깝게 줄인다"

### F3. 송장 관리 (핵심 도메인)

| ID | 기능 | 상태 | 설명 |
|----|------|------|------|
| F3.1 | 송장 목록 조회 | 🟢 | 상태/클라이언트/검색어 필터, 정렬, 페이지네이션 |
| F3.2 | 송장 생성 | 🟢 | 클라이언트 연결, 항목(items) 추가, 금액/마감일 설정 |
| F3.3 | 송장 상세 조회 | 🟢 | 송장 정보 + 클라이언트 + 항목 + 리마인더 상태 |
| F3.4 | 송장 수정 | 🟢 | draft 상태에서만 수정 가능 |
| F3.5 | 송장 삭제 | 🟢 | draft 상태에서만 삭제 가능 |
| F3.6 | 송장 발송 | 🟢 | 이메일 발송 + 결제 링크 생성 + 리마인더 예약 (원스톱) |
| F3.7 | 결제 완료 처리 | 🟢 | 수동 또는 Paddle Webhook으로 paid 처리 + 리마인더 취소 |
| F3.8 | 연체 자동 전환 | 🟡 | ReminderScheduler에서 매시간 스캔, overdue 전환 |
| F3.9 | PDF 다운로드 | 🟢 | ChromicPDF로 송장 PDF 생성 및 다운로드 |
| F3.10 | 송장 번호 자동 채번 | 🔴 | 현재 수동 입력, 자동 채번 미구현 |

**송장 상태 머신**:
```
draft → sent → overdue → paid
               sent → paid (조기 결제)
```

**발송 시 연쇄 동작** (F3.6):
1. 상태 → `sent` + `sent_at` 기록
2. Paddle 결제 링크 자동 생성
3. 3단계 리마인더 자동 예약 (D+1, D+7, D+14)
4. 클라이언트에게 송장 이메일 발송
5. PubSub으로 대시보드 실시간 반영

**결제 완료 시 연쇄 동작** (F3.7):
1. 상태 → `paid` + `paid_at` 기록
2. 예약된 리마인더 전부 취소 (Oban Job 포함)
3. PubSub으로 대시보드 실시간 반영

**비즈니스 규칙**:
- Free 플랜: 월 3건 제한
- Starter/Pro: 무제한
- draft가 아닌 송장은 수정/삭제 불가

## 3. 사용자 흐름 (User Flows)

### Flow 1: 신규 사용자 온보딩
```
1. 랜딩 페이지 방문 (/welcome)
2. "무료로 시작하기" 클릭
3. Google OAuth 또는 이메일로 가입
4. 대시보드로 리디렉트
5. 첫 송장 업로드 유도
```

### Flow 2: 송장 업로드 → 자동 결제 (Happy Path)
```
1. 대시보드에서 "Upload" 클릭
2. PDF 드래그 앤 드롭
3. AI가 데이터 추출 (로딩 표시)
4. 추출 결과 확인/수정
5. "Create Invoice" → 송장 생성 (draft)
6. "Send" → 이메일 발송 + 리마인더 예약
7. [D+1] 클라이언트에게 리마인더 #1 발송
8. 클라이언트가 "Pay Now" 클릭 → Paddle 결제
9. 결제 완료 → 자동으로 paid + 리마인더 취소
10. 대시보드에 실시간 반영
```

### Flow 3: 미결제 에스컬레이션 (Unhappy Path)
```
1. [D+1] Step 1 리마인더 → 반응 없음
2. [D+7] Step 2 리마인더 → 이메일 열람 (추적)
3. [D+14] Step 3 최종 경고 → 클라이언트 결제
4. 결제 완료 → 리마인더 자동 종료
```

### Flow 4: Free → Starter 업그레이드
```
1. 월 3건 송장 소진 알림
2. "Upgrade" 버튼 클릭
3. Paddle 결제 페이지
4. 결제 완료 → Webhook → 플랜 자동 업그레이드
5. 무제한 송장 + AI 리마인더 활성화
```
